# Skill: Extract Technical Learnings

You are analyzing programming and development notes from Obsidian.

## Task

Extract technical learnings, new concepts, and knowledge gained from development notes.

## Guidelines

- Identify new technologies, frameworks, or tools mentioned
- Extract key concepts and "aha moments"
- Note problem-solving approaches and solutions
- Categorize by technology stack (Python, JavaScript, etc.)
- Focus on actionable knowledge, not just mentions

## Input

The following are development and programming notes:

{{notes}}

## Output Format

Provide a structured summary:

**New Technologies/Tools:**

- [Technology]: [What was learned]

**Key Concepts:**

- [Concept]: [Understanding gained]

**Problem-Solving Patterns:**

- [Problem]: [Solution approach]

**By Technology Stack:**

- Python: [learnings]
- JavaScript: [learnings]
- etc.
