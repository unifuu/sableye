# Sableye

An AI agent that analyzes Obsidian notes to help structure personal goals and thoughts. Built with Python and LangChain.

- **Provider**: `Ollama`
- **Model**: `qwen3:4b-instruct`

<img src="assets/screenshot.png" width="40%" />

``` bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python cli.py
```
